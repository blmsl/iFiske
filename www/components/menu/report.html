Har du hittat en bugg eller problem med appen?

Vi ser gärna att du rapporterar in buggar på GitHub

Du måste ha ett användarkonto och logga in på GitHub för att skriva in en rapport - men då kommer du också få feedback när din rapport behandlas! Att göra ett konto på GitHub är gratis.

Om du ej kan använda GitHub kan vi ta emot felrapporter på app@ifiske.se - Se till att beskriva felet noga och ange också vilken telefontyp du kör på. Felrapporter av typen "det funkar inte" kan vi tyvärr inte göra så mycket med.

Tack så mycket!
